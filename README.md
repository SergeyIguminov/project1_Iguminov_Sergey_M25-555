Labyrinth Game

Текстовая приключенческая игра в жанре "лабиринт", где игрок исследует таинственные комнаты, решает загадки и собирает предметы, чтобы найти сокровище.

Особенности

- Текстовый интерфейс - классический геймплей в стиле MUD
- Система загадок - уникальные головоломки в каждой комнате
- Случайные события - динамический игровой процесс
- Система инвентаря - сбор и использование предметов
- Нелинейный лабиринт - множество путей к победе

Установка

Предварительные требования
- Python 3.8+
- Poetry

Установка зависимостей
# Клонируйте репозиторий
git clone https://github.com/SergeyIguminov/project1_Iguminov_Sergey_M25-555.git
cd project1_Iguminov_Sergey_M25-555

# Установите зависимости через Poetry
poetry install

# Или используйте Makefile
make install
